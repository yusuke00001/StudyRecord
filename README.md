### StudyRecord

■ アプリ概要

StudyRecordは、「間違えた問題を適切なタイミングで確実に復習する」ための学習管理アプリです。

科目 → 教材 → 問題単位で学習記録を管理し、
復習対象を構造的に整理できる設計になっています。

単なる教材管理ではなく、
**復習の最適化を支援すること**を目的としたアプリです。

---

■ 開発背景

学習を進める中で最も重要なのは
一度間違えた問題を確実に復習することだと感じました。

しかし従来の管理方法（問題集へのチェック記入など）では、
	- 復習を忘れてしまう
	- 復習までの間隔が空きすぎる
	- どの問題を優先すべきか分からない
といった課題がありました。

特に、復習タイミングが遅れることで記憶定着率が大きく低下することを実感し、
エビングハウスの忘却曲線に基づく復習管理の必要性を感じました。

そこで本アプリでは、「どの科目の」「どの教材の」「どの問題を」「いつ学習したのか」を構造的に管理できる仕組みを実装しています。
